Intro to Responsive Design


Prior to the responsiveness of modern frameworks, people explored the vast open webscape without an expandable net. Imagine a time when surfing the web required a magnifying glass, multiple scroll bars, and patience the size of, well, the web itself.

This is where Responsive Web Design (RWD) comes in. RWD has allowed us to rocket forward in design for the web. Due to the ever changing demands on websites, and the continuous size changes of viewing devices, it has become necessary to create websites that can be viewed across multiple platforms. Rather than building new website for each device, as was often the case before RWD, there have been new developments that combine JavaScript and CSS with HTML to allow for cross-platform viewing.

So what are the building blocks of a responsive web design? Amongst other things it consists of a mix of flexible grids, layouts, and smart use of CSS media queries. These are exciting times for both web developers and graphic designers. It allows each component to be well thought out, and vastly increases the amount of dynamic, creative, and intelligent websites.
